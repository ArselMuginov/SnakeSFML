# SnakeSFML
Simple snake game using C++ and SFML

## Status
- Currently game logic is not done
- GUI is mostly done
- Needs polishing

## Dependencies
- C++17
- SFML 2.5.1 x64
  - System module
  - Window module
  - Graphics module
- [gpakosz/whereami](https://github.com/gpakosz/whereami) (included in code)
